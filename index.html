What's up with this weird website?

Remember when most (all?) websites looked basically like this? I don't really, by the time my house got a computer, around 2002, I was 13 or 14. I spent a lot of time playing Midtown Madness and learned the street networks of Chicago, San Francisco and London.

If you're intested in the simplicity of a website built exclusively with HTML and CSS check out CSS Zen Garden, which has some cool examples of the power of web development's two most basic tools. These, along with C, and SQL are really amazing in their simplicity to power ratio.

My plan is to build multiple versions of my site, using different tools. Currently I'm working on version 1, using only html and css. I haven't made a website without bootstrap before. I also haven't made a multi-page site without using flask or express to use the same template for the various content.

The next version will definitely include a CSS theme from somewhere else. I'm not very good at design stuff like that. I'll probably structure it as a single page continuous scroll type of thing since those are popular. That doesn't scale very well though. I'll have to think about the structure a little. Maybe a scroll page for the various topics, which would each be an intro to the full scroll page for a given topic.

I'll then set up a site using a real backend. Not sure how that will work with the github.io hosting. Will probably do an express version and a flask version... We'll see.

Finally, I want to use React a little but would need a real use case for it.
